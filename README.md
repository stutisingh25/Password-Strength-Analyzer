# Password Strength Analyzer

A Flask-based web application that analyzes password strength in real time.

## Features

- Password strength analysis
- Strength meter
- Password suggestions
- Password hashing using bcrypt
- Prevents password reuse
- SQLite database
- Responsive UI

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- bcrypt

## Installation

```bash
pip install -r requirements.txt
python app.py
